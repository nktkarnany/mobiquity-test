About 1. The tabs are placed in the menu bar. Swipe screed from left edge to open the menu bar. 2. Menu bar consists of four tabs - home, articles, settings and about. 3. Home tab is the list of categories for which the news is to be fed. Selecting any one of it saves it in the local storage and displays the articles for that particular category. 4. Articles tab displays the list of articles fetched from the reddit api using the category and limit provided by the you. 5. Settings tab allows you set the limit of articles to be fetched.
