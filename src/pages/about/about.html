About How I Work - 1. The tabs are placed in the menu bar. Swipe screen from left edge to open the menu bar. 2. Menu bar consists of four tabs - home, articles, settings and about. 3. Home tab is the list of categories for which the news is to be fetched. Clicking on any one of it saves it in the local storage and displays the articles for that particular category. 4. Articles tab displays the fetched articles from the reddit api. While making the api call it uses the category and limit saved in the local storage. 5. Settings tab allows you set the limit for number of articles to be fetched.
